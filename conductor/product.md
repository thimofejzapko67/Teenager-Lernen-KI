# Product: ClawAcademy

## One-Liner

Kostenlose KI-Entwicklungs-Plattform für Teenager (13-19 Jahre) mit AI-First Ansatz, Rank-System, und Sponsoring.

---

## Problem Statement

Teenager wollen KI-Tools nutzen und Apps entwickeln, aber:
- Tutorials sind zu kompliziert oder langweilig
- Kein strukturiertes Lernpfad für KI-Entwicklung
- Keine Verbindung zur Industrie (Praktika, Jobs)
- Fachchinesisch schreckt ab
- Keine Feedback-Loop für Code

---

## Solution Approach

**ClawAcademy** ist eine Lernplattform die:
- KI-Tools und Prompting lehrt (nicht Code-Syntax)
- Kurze, gamified Lektionen (max 15 Min)
- Rank-System mit XP und Belohnungen
- Sponsoren-Plattform für Firmen-Talente
- Community mit Hackathons und Mentoren

**AI-First Philosophy:**
Die KI schreibt den Code, der User lernt Prompting und Konzepte.

---

## Target Users

### Primary: Leo (13-16)
- Null Vorkenntnisse
- Spielt Games (Minecraft, Roblox)
- Will Spass haben und Dinge bauen
- 15-30 Min/Day verfügbar

### Secondary: Mira (15-18)
- HTML/CSS Grundlagen
- Will Portfolio für Praktika
- 1-2 Stunden/Day

### Tertiary: Tim (16-19)
- Fortgeschritten
- Will Startup gründen
- 3+ Stunden/Day

---

## Core Features

### 1. Rank & Progression System
- 6 Ränge: Novice → Coder → Developer → Architect → Master → Legend
- XP aus Lektionen, Projekten, Events, Streaks
- Leaderboard (Global, Weekly, Monthly, Friends)
- Ränge können verloren werden (Inaktivität)

### 2. Lektionen (Micro-Learning)
- 15 Minuten pro Lektion
- Struktur: Konzept → Prompt Example → Guide → Quiz
- 6 Kategorien: KI-Basics, Web-Dev, Mobile-Dev, KI-Agenten, AGI-Sicherheit, Security
- Copy-paste Prompts für KI-Tools

### 3. Sponsoren-System
- Top 3 bekommen Premium Sponsoren
- Top 100 sind scoutbar für Firmen
- Firmen durchsuchen Profile nach Skills/Rank
- Sponsor-Pakete: Bronze (€500) → Platinum (€15,000)

### 4. Community
- Discord Integration
- Hackathons (Solo + Team)
- Mentoren-Programm
- Projekt-Upload mit Community-Rating

### 5. Gamification
- Achievements (Badges)
- Daily Streaks mit Bonus XP
- Story Mode (Missionen)
- Events mit Rewards

## Product Roadmap

### Phase 1: MVP (Months 1-3)
- Lernen: KI-Basics, Web-Dev Intro
- Platform: Auth, Profiles, XP, Leaderboard
- Content: 30 Lektionen
- Launch: Beta mit 50 Usern

### Phase 2: Growth (Months 4-9)
- Content: 50+ Lektionen
- Platform: Sponsors Marketplace, Projects
- Community: Hackathons, Mentors
- Monetization: Erste Sponsoren

### Phase 3: Scale (Months 10-24)
- Content: 100+ Lektionen
- Platform: PWA, Offline Mode
- Community: 10k+ User
- Expansion: EU

### Phase 4: Ecosystem (Months 25+)
- Mobile: Native iOS/Android Apps
- Professional: Zertifikate, Jobs
- Enterprise: B2B Platform

## Differentiators

1. **AI-First** - KI schreibt Code, nicht User
2. **Gamified** - XP, Ränge, Achievements
3. **Sponsorship** - Direkte Verbindung zu Firmen
4. **Community** - Hackathons, Mentors, Discord
5. **Micro-Learning** - 15-Minuten Lektionen
6. **Teenager-Fokus** - Kein Fachchinesisch, echt

---

## Business Model

**User:** Komplett kostenlos (für Teenager)

**Revenue Streams:**
1. Sponsorships (Firmen zahlen für Sichtbarkeit/Rekrutierung)
2. Affiliate Links (Hosting, Tools, KI-Dienste)
3. Event Sponsorships (Hackathons)
4. State Funding (Bildungsprojekte)

**Cost Structure:**
- Hosting (Vercel, Supabase)
- Content Creation
- Community Management
- Legal/Compliance
