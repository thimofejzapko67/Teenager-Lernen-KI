## Project Overview

**ClawAcademy** is a free AI learning platform for teenagers (13-19 years) that teaches AI tool usage and app development through an AI-first approach. The AI writes code; users learn prompting.

### Key Features
- Rank & XP progression system
- 15-minute micro-lessons
- Sponsorship marketplace connecting talent with companies
- Community with hackathons and mentors
- Gamified learning with achievements

### Tech Stack
- **Frontend**: Next.js 15.2, React 19, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage, Realtime)
- **Deployment**: Vercel
- **AI**: Claude (primary), DeepSeek (secondary)
