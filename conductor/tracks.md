## Track States

| State | Description |
|-------|-------------|
| **Pending** | Track defined, not started |
| **In Progress** | actively being worked on |
| **In Review** | Deliverables ready for review |
| **Completed** | All deliverables done, deployed |
| **Blocked** | Waiting on dependency |
